Scora

Scora is a board game scoring and statistics application designed to help players track their games, record scores, and analyze their play history over time.

Main features

  • Add board games to record played sessions and scores
  • Track game results and keep a full history of matches per game
  • View personal statistics and performance trends
  • Add friends and compare statistics between players
  • Social and advanced analytics features (planned)

BoardGameGeek is used as a reference data source to identify board games and retrieve public metadata when a user adds a game to Scora.
