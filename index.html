Scora

Scora is a board game collection management application designed for players who want to organize, track, and enrich their personal game library.

Main features

  • Personal board game collection management
  • On-demand import of games from BoardGameGeek
  • Personal notes, ratings, and game status
  • Social and discovery features (planned)

Scora uses BoardGameGeek as a reference data source to identify and retrieve public metadata about board games when explicitly requested by the user.

Powered by BoardGameGeek
